# YACT (Yet Another Clash Toolkit)


[![Go Report Card](https://goreportcard.com/badge/github.com/ethannhu/yact)](https://goreportcard.com/report/github.com/ethannhu/yact)
![GitHub License](https://img.shields.io/github/license/ethannhu/yact)
![GitHub Go version](https://img.shields.io/github/go-mod/go-version/ethannhu/yact)
![GitHub release (latest by date)](https://img.shields.io/github/v/release/ethannhu/yact)

<br />

**YACT** is a minimal CLI control panel based on Mihomo API.
It supports **Unix Pipeline** and **Interactive TUI**

---

</div>

YACT is developed with minimalism in mind, inspired by *suckless*.
- No extra config file
- No modification on Mihomo API 
- Self-contained usage tips


---

## Deployment

### Go Module
```bash
go install [github.com/ethannhu/yact@latest](https://github.com/ethannhu/yact@latest)
```
### Prebuilt Binary
``` bash
# Download from release
tar -zxvf yact_Linux_x86_64.tar.gz
chmod +x yact
sudo mv yact /usr/local/bin/
```
